## Автоматизация тестирования на Python + Selenium

#### Структура
Материал организован по урокам (lesson_01, lesson_02, ...).
Каждый урок содержит несколько практических задач в формате ```.md``` файла и их решение – ```.py``` файл.


- `README.md` - описание проекта
- `lesson_01/` – модуль 1: Поиск элементов с помощью Selenium WebDriver
- `lesson_02/` – модуль 2: Полезные методы Selenium
- `lesson_03/` – модуль 3: Тестовые фреймворки
- `requirements.txt` - используемые зависимости

Курс развился еще на два репозитория:

- [Здесь](https://github.com/mrnvnst/stepik-ui-language-autotesting) хранятся автотесты для интернет-магазина с учетом разных языков интерфейса.
- А [здесь](https://github.com/mrnvnst/stepik-pom-autotesting) можно посмотреть более расширенный сценарий тестирования того же магазина, но уже с применением Page Object.
